# Radar-Signal-Processing

## Description
This project is a **midterm assignment for Math III (Linear Algebra)** that explores the application of mathematical concepts in real-world engineering systems. The focus of the project is on **target detection using FMCW radar** combined with the **Fast Fourier Transform (FFT)**.

## 📡 Project Overview
This project simulates an **FMCW radar system** and applies **FFT-based signal processing** to detect targets.

### 🎯 Capabilities
- Distance estimation  
- Velocity detection  
- Angle estimation  

---

## 🧠 Key Concepts
- **FMCW Radar** → Continuous chirp signal  
- **FFT (Fast Fourier Transform)** → Time → Frequency conversion  
- **Signal Processing** → Extract hidden information  

---

## System Workflow
```
Transmit Signal → Receive Echo → Mix → Data Cube
        ↓
     Windowing
        ↓
   FFT Processing
        ↓
Range → Doppler → Angle
        ↓
   Visualization
```
